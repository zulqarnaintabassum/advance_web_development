🔗 Task 3: Shared Counter Service

Both components inject the same CounterService singleton. Any change in one immediately appears in the other.

⚙️
How Dependency Injection Works Here

Angular's DI system creates a single CounterService instance (providedIn: 'root'). Both CounterDisplayOneComponent and CounterDisplayTwoComponent receive a reference to that same instance — so currentCount is truly shared.
